# 🌙 Sleep Love - Atmospheric Meditation & Sleep App

**Sleep Love** is a high-fidelity, premium sleep meditation application designed to help users relax, focus, and achieve deep sleep through immersive atmospheric soundscapes. Developed by **Questra**, the app features a stunning glassmorphic UI, a robust offline caching engine, and a dynamic mixing system.

---

## ✨ Key Features

- **🎧 Immersive Soundscapes:** High-quality rain, nature, and meditation sounds.
- **🧪 Liquid Mixer:** Mix multiple sounds simultaneously to create your perfect atmosphere.
- **🔋 Premium Timer System:** Earn premium access by watching ads, with a real-time countdown progress bar.
- **🎨 Dynamic Themes:** Seamlessly switch between Light and Dark modes with persistent saving.
- **📥 Smart Caching:** Automatic "Download-then-Play" engine for a zero-lag experience.
- **🚶‍♂️ Interactive Tour:** Guided onboarding for new users with a spotlight tour.
- **📱 Responsive Design:** Fully optimized for all Android devices with liquid glass aesthetics.

---

## 🛠️ Built With

- **Framework:** [Flutter](https://flutter.dev/) 3.11+
- **State Management:** [Riverpod](https://riverpod.dev/)
- **Database:** [Firebase Realtime Database](https://firebase.google.com/)
- **Animations:** [Flutter Animate](https://pub.dev/packages/flutter_animate)
- **Audio Engine:** [Just Audio](https://pub.dev/packages/just_audio)
- **Monetization:** [Unity Ads](https://unity.com/products/unity-ads)

## 🚀 Getting Started

### Prerequisites

- Flutter SDK installed.
- Firebase project configured with `google-services.json`.
- Unity Ads Game IDs configured in `AdService`.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/RukshanAmodya/Sleep-Love.git
   ```

2. **Install dependencies:**
   ```bash
   flutter pub get
   ```

3. **Run the app:**
   ```bash
   flutter run
   ```

## 📄 Privacy & Terms

Sleep Love respects user privacy. No audio data is collected. Premium status is synchronized via Firebase. Caching is handled locally for performance.
